1) Clone le fichier:
git clone "lien du fichier (https)"

2) pr ajouter un fichier:

cd "adresse du dossier"
git add "nom du fichier"
git commit -m "message"
git push

3) pr mettre a jour le fichier

cd "adresse du dossier"
git pull

4) Info utiles

savoir si on est à jour par rapport au serveur: git status
